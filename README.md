# ⚡ Harry Potter RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that answers questions from the Harry Potter books using Large Language Models and semantic search.

---

## Features

- Retrieval-Augmented Generation (RAG)
- Semantic Search
- Vector Database
- PDF Knowledge Base
- Context-Aware Responses
- Large Language Model Integration

---

## Tech Stack

- Python
- LangChain
- FAISS / ChromaDB
- Hugging Face Embeddings
- Google Gemini / OpenAI
- Streamlit
- PyPDF
- Sentence Transformers

---

## Architecture

PDF
↓
Text Chunking
↓
Embeddings
↓
Vector Database
↓
Retriever
↓
LLM
↓
Final Answer
